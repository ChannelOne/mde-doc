Class SearchBox

Searchbox doesn't reply on the common view component in this project. it just replies on the HTML dom api to handle.

Use SearchBox.GetOne() to get a gobal SearchBox instance, using new to create an instance is not working.

There are two events to listen

  • Search
  • Replace
